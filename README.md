# Análisis y Diseño de Sistema – HOMEFIX

Caso práctico de análisis de requisitos y diseño de un sistema para la gestión de pedidos de reparación y mantenimiento del hogar.

## Alcance del trabajo
- Análisis de requisitos a partir de una consigna inicial
- Elaboración de casos de uso
- Diagramas de flujo
- Diagramas de secuencia
- Diagramas de clases

## Notas
El sistema no contempla cálculo de precios ni costos. Se considera una posible extensión futura con un módulo de suscripción.
